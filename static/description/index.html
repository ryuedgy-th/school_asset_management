  • Teacher: View own assignments only
  • Portal User: Parents can view student assignments

Installation

  1. Copy module to Odoo addons directory
  2. Update Apps List
  3. Install "School Asset Management"
  4. Configure categories and locations
  5. Assign user groups

Optional Dependencies

For QR code generation:

pip install qrcode[pil]
